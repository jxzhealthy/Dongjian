# 贡献指南 (Contributing to Dongjian)

欢迎参与 **Dongjian (洞见)** 的建设！你的每一份贡献都能帮助开发者更好地理解代码。

## 🛠️ 如何开始

1. **Fork 本仓库**：点击右上角的 Fork 按钮。
2. **克隆到本地**：`git clone https://github.com/your-username/DongJian.git`
3. **安装依赖**：进入目录并运行 `npm install`。
4. **创建分支**：`git checkout -b feature/YourFeatureName`。

## 💻 开发规范

*   **代码风格**：请遵循项目现有的 TypeScript 编码规范。
*   **提交信息**：建议使用语义化提交，例如 `feat: add python parser` 或 `fix: resolve cytoscape loading error`。
*   **测试**：在提交 PR 前，请确保 `npm run build` 能成功通过。

## 📝 文档维护

如果你修改了核心逻辑，请同步更新 `docs/` 目录下的相关设计文档或 `README.md`。

## 🐛 报告问题

如果你发现了 Bug 或有新功能建议，请在 [Issues](https://github.com/your-username/DongJian/issues) 中提出。

感谢你的支持！
