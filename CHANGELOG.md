# 更新日志 (Changelog)

所有重要的项目变更记录在此。

## [0.1.0] - 2026-07-03

### ✨ Added
- 初始版本发布：支持 TypeScript/JavaScript 代码库扫描。
- 集成 Tree-sitter 实现确定性结构解析。
- 接入 Qwen3.7-max 实现 AI 语义分析与架构分层。
- 内置基于 Cytoscape.js 的交互式 Web Dashboard。
- 实现基于 SHA-256 哈希的增量同步机制。
- 完善项目文档：包含 README、架构设计及贡献指南。

### 🛠️ Technical
- 采用 Node.js/TypeScript 全栈技术栈。
- 优化边过滤逻辑，解决 Cytoscape 渲染崩溃问题。
- 实现本地化库引用，确保离线环境可用性。
